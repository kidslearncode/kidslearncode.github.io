Scratch

Scratch is the world's largest coding community for children and a coding language with a simple visual interface that allows young people to create digital stories, games, and animations.

Python

Python is a programming language that lets you work quickly and integrate systems more effectively. In Kids Learn Code, we use a web-based platform, Replit, to enable students to use Python more easily, without installation.
